# Isuzu — Marketing dashboard (frontend)

Vite + React application for marketing plans, budgets, calendar, charts, and admin workflows.

## Documentation

Feature and architecture documentation lives in **[`docs/`](./docs/README.md)**.

## Quick start

```bash
npm install
npm run dev
```

Optional: set `VITE_API_BASE_URL` in `.env` (see `docs/01-application-overview.md`).

## Scripts

- `npm run dev` — development server  
- `npm run build` — production build  
- `npm run lint` — ESLint
